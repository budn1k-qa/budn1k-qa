1.Работа с базами данных
Создать папку на локальном компьютере, вложить в нее нужные файлы
Открыть VSCode - нажать Source Control
Затем нажать Open Folder и выбрать свою папку
Далее нужно инициализировать репозиторий - открыть терминал и прописать $ git init
Добавить файлы $ git add .
На сайте гитхаб создаем новый репозиторий databases
В терминале коммитим наши изменения
Добавляем удаленный репозиторий $ git remote add origin https://github.com/username/reponame.git
Закомиттил изменения раньше времени (потому что создал репо с файлом ридми, которого на локальном пк не было, 
и попытался прописать pull и push: гит ругается - разные истории
Поэтому игнорим командой $ git pull origin master --allow-unrelated-histories (данных у меня удаленно все равно никаких нет, кроме README.md)
$ git push -u origin master
Репозиторий с данными готов.
